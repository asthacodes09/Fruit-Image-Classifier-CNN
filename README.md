# Fruit Image Classifier using CNN

## Description
This project is a deep learning-based image classification system that classifies fruits into Apple, Banana, and Orange categories using Convolutional Neural Networks (CNN).

## Technologies Used
- Python
- TensorFlow
- Keras
- CNN

## Dataset Structure
fruit_dataset/
- train/
  - apple
  - banana
  - orange
- test/
  - apple
  - banana
  - orange

## Result
The model successfully classifies fruit images with good accuracy.
